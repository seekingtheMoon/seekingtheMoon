brief introduction

使用 Git 版本控制，将本地仓库同步到 GitHub 上

创建 GitHub 仓库

登录 GitHub 账户，New repository

填写仓库名称、描述等信息，并选择是否公开或私有。

点击 "Create repository" 按钮创建仓库。

建议不要勾选自动生成 README 文件，不然远程仓库和本地推送仓库会出现两个分支，我目前还不知道这种情况有什么好的办法解决（有猜测，但还未尝试）
