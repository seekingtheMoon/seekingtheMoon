Win10搭建Python环境

发表于 2022-02-23 更新于 2022-03-08 分类于 Notes 本文字数： 405 阅读时长 ≈ 1 分钟

教程基于《Python编程 从入门到实践（第二版）》

Python: 3.10.2

Windows10

  1. 访问Python官网 https://www.python.org/（浏览器搜索也一样，注意官网标识，不要进错了）

  2. 下载最新版的Python即可

  3. 打开下载好的安装包，确保圆圈处的方框打上勾，然后可以直接点击1处进行安装，想要自行安装点击2处

  4. 等待安装结束后，可能会弹出“Disable path length limit”，点击它，禁用系统的 Path 长
    度自动限制，可以规避很多的麻烦

  5. 完成安装（Setup was successful）

  6. 打开cmd，输入python，然后回车，如果出现了 Python 提示符（>>>），说明Windows找到了你刚安装的Python版本
